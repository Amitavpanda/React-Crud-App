* Integration with JSON Server for simulating a RESTful API

## Setup
To run this project locally 
 
 1. Clone the repository
```
git clone https://github.com/Amitavpanda/React-Crud-App.git
```
2. Install the dependencies
```
npm install
```
3. Run the app
```
npm start
```
